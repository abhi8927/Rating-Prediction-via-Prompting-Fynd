# Yelp Rating Prediction & AI Feedback System

## Assignment Overview

This repository contains the implementation for a two-part assignment:

1. **Task 1**: Rating Prediction via Prompting - Jupyter notebook with prompt engineering experiments
2. **Task 2**: Two-Dashboard AI Feedback System - Full-stack web application

## Project Structure

```
├── task1/                    # Rating prediction notebook
│   ├── rating_prediction.ipynb
│   ├── data/
│   ├── prompts/
│   └── utils/
├── task2/                    # Web application
│   ├── backend/              # Flask API
│   └── frontend/             # React application
├── report/                   # Project report
└── README.md
```

## Setup Instructions

### Prerequisites
- Python 3.8+
- Node.js 16+
- Git
- Google Gemini API key

### Installation

1. Clone the repository
2. Install backend dependencies: `cd task2/backend && pip install -r requirements.txt`
3. Install frontend dependencies: `cd task2/frontend && npm install`
4. Set up environment variables (see `.env.example`)
